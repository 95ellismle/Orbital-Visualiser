Quick Start

A standard workflow for using the movie maker normally consists of 2 steps. A calibration step and a movie creation step.

However, before running the code you need to remember to make sure the path is pointing to the correct data. You can set this in the settings file by inputting the variable path:

path = "data path" # Path to CP2K output data

The Calibration Step:

The calibration step simply creates a visualisation for one of the frames in your image. This is useful if you just want a single image for a poster or a paper; it is also useful for calibrating settings before setting the code running on a long visualisation. Simply set the variables:

*calibrate* = 'y' # Turns on the calibration step
*load_in_vmd* = 'y' # Loads the data in VMD

This will load 1 frame of your image in VMD, so you can adjust the view to suit you. The code should save all rotations, translations and scalings to save you having to input these into the settings file. However other settings may need to be adjusted. For example, if the isosurface appears to be truncated then you need to increase the *bounding_box_scale*, if the isosurface is too large or small you will need to adjust the *isosurface_to_plot*

Creating the Movie:

Once you are sure you are happy with the settings you have you can run your visualisation and create the movie. In most cases you can simply turn off the calibration (*calibrate* = 'n') and tell the code how long to run the simulation for e.g:
*start_time* = 0 # Start at time 0 (step 0)
*end_time* = 'all' # Iterate over the whole simulation data
*stride* = 2 # Work with every 2nd step

From here you can also set the *movie_length* and the *movie_format*. For all the settings, see the Settings Directory (gray button above).
