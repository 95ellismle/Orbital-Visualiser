Setting Default Value Description Input Type

neg_real_iso_col (0,0,1) The color of negative real isosurfaces ['list [red, green, blue]']
neg_imag_iso_col (0,0.4,1) The color of positive imaginary isosurfaces ['list [red, green, blue]']
pos_imag_iso_col (1,0.4,0) The color of negative imaginary isosurfaces ['list [red, green, blue]']
density_iso_col (0.3,0.32,0.3) The color of the isosurface in a density visualisation ['list [red,green,blue]']
carbon_color black The color of the Carbon atoms ['string']

background_color (1,1,1) The color of the background ['string', 'list (rgb values e.g. [1,0,0])']
neon_color yellow The color of the Neon atoms ['string (VMD colors)']
mol_style CPK The visualisation style of the molecules (see VMD) ['string']
mol_material Edgy The visualisation material used for the molecules (see VMD) ['string']
