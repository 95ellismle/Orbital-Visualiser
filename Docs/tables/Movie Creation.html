Movie Creation

Setting Default Value Description Input Type
"title" None This will change the name of the movie and the folder it is saved in. Will save the movie in ./img/Title/
  • string
end_step 'all' Which step to iterate up to for the movie
  • all
  • integer
stride 1 What stride to use to iterate with for the movie (a stride of 2 would render every 2nd frame)
  • integer
start_step 0 Which step to start iterating on
  • integer (zero is the first)
verbose_output 'no' Shows lots of info about what is happening
  • yes
  • no
movie_length 6 The length of the finished movie (will adjust the framerate to match the time)
  • integter
movie_format "mp4" The format of the created movie
  • mp4
restart_vis 'yes' Will detect if you are using the same title as any in the img/ folder. If you are it will only visualise the steps not completed and stitch all imgs together into a movie at the end
  • yes
  • no
