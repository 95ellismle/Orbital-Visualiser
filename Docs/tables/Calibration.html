Calibration

Setting Default Value Description Input Type
calibrate 'yes' Performs a single `calibration' step and save the image in the folder ./img/Calibration/
  • yes
  • no
calibration_step 10 Chooses which (nuclear) timestep to perform the calibration on
  • integer
show_img_after_vmd 'no' Loads the calibration img after the code has ran
  • yes
  • no
load_in_vmd 'no' Loads the calibration img in VMD. This is useful for setting the position of the image as the code will save any rotations, scalings and translations performed within VMD
  • yes
  • no
isosurface_to_plot 0.003 Which isosurface to plot during the simulation, a smaller number will show a larger isosurface
  • float
